## Getting Started

### Prerequisites

- Node.js 18+ 
- pnpm (recommended) or npm

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd youtupe-transcrtip-v2
```

2. Install dependencies:
```bash
pnpm install
```

3. Run the development server:
```bash
pnpm dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. Paste a YouTube URL in the input field
2. Click "Get Transcript" or press Enter
3. The app will automatically detect available languages for the video
4. Select your preferred language from the dropdown (shows only available languages)
5. Choose output format (Text/SRT/JSON)
6. Copy or download the transcript

**Note:** Languages marked with "(Auto)" are auto-generated by YouTube.

## Supported URL Formats

- `https://youtu.be/VIDEO_ID`

## Deployment

### Deploy to Vercel

1. Push your code to GitHub
2. Import your repository in Vercel
3. Vercel will automatically detect Next.js and configure the build
4. Deploy!

The app is optimized for Vercel deployment.

## Environment Variables (Optional)

### YOUTUBE_API_KEY (Optional)
For better language detection, you can use YouTube Data API v3:

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Create a new project or select existing one
3. Enable YouTube Data API v3
4. Create API Key
5. Add to `.env.local`:

```bash
YOUTUBE_API_KEY=<your-api-key>
```

**Note:** Without API key, the app will still work but will use fallback method to detect available languages from YouTube page.

## Known Limitations

- Some videos may not have transcripts available
- Transcripts are only available in languages that the video creator provided
- Rate limiting may apply for excessive requests
